# Books Searching

## Brief Description

In this project, google books is helping you searching the book by insert the book name.

## Prerequisites
--> NodeJs >= 16.11.19
## Install project

### `npm install`

## Start the project
### `npm start`
Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
